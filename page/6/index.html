如何从YouTube上下载视频

发表于 2016-04-15 分类于 Learning

介绍2个下载YouTube视频的方法。

savefrom.net

savefrom.net是一个视频下载网站，应该很多人用过了吧！支持很多主流视频网站，而YouTube也是其中之一，在国内的速度还是不错的！而且很不错的是他提供一些插件和软件，比如chrome和Mozilla Firefox等直接安装插件在YouTube、Facebook、 Vimeo、VK.com网站就可以下载了！
官网：http://en.savefrom.net/

使用方法也是很简单，进网站输入要下载的视频网址，点右边那个箭头按钮就好了！

同时他提供一些插件（其实是脚本），比如我在Chrome中安装了插件，在YouTube视频下面就会出现这个选项，就可以直接下载了。

Chrome使用步骤：

1.安装Tampermonkey插件，并启用。

2.打开helper.user.js，并点击安装。

3.重启浏览器，再进YouTube试一下吧！

Mozilla Firefox使用步骤：

1.下载扩展，并安装。

2.重启浏览器，再进YouTube试一下吧！

如果你不用chrome和Mozilla Firefox那就去这里看一下有没有你的浏览器对于的插件脚本。

ClipConverter

这个网站跟savefrom.net的区别是，他可以在线视频、音频转码，而且直接就可以下载1080P的是，而savefrom.net下载1080P的需要下载他的软件才可以。
官网：http://www.clipconverter.cc/

使用方法也是很简单，进网站输入要下载的视频网址，点右边那个Continue按钮就好了！

然后选择视频清晰度，也可以在下面选择转码并下载，选完点Start！按钮

这时候直接就点Download下载视频吧！

这里视频下载网站还挺多的，我只是简单的列举了两个，如果有什么插件、脚本、网站推荐请在下面留言，谢谢！

学习帮助建议

发表于 2016-04-13 分类于 Learning

1、善用Google（Google如果被墙可以使用ss翻墙）。基本上你遇到的80%以上的技术疑问，都能在Google上找到答案，并且你提出的疑问基本上都有人在一个叫Stack Overflow的神奇网站上提问过并且有人做了很好的回答。记住：想要在技术的道路上走得更远，就不要使用某度。

2、提高英文阅读能力。基本上所有计算机技术都是外国人发明的，所以英文的文档和说明都比较全面和精细

3、了解某个概念。例如：什么是Javascript?先wiki后百度百科

4、了解无固定答案的问题。例如：什么是良好的编码习惯？建议在知乎上搜索或提问

5、遇到问题先独立思考，和同组成员一起交流讨论，多问前辈
